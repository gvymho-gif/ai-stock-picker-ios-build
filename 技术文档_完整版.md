### 3.1 初始化服务器


### 3.3 创建 systemd 服务

**API 服务** (`/etc/systemd/system/aistock-api.service`):
```ini
[Unit]
Description=AI Stock Picker API
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/root/backend
Environment=AUTH_TOKEN=blueprint_aistock_2026_123456
Environment=PORT=8000
ExecStart=/root/backend/venv/bin/python -m uvicorn main:app --host 0.0.0.0 --port 8000
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
```

**监控服务** (`/etc/systemd/system/aistock-monitor.service`):
```ini
[Unit]
Description=投资组合实时监控（止盈止损）
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/root/backend
ExecStart=/root/backend/venv/bin/python /root/backend/portfolio_monitor.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

启用命令：
```bash
systemctl daemon-reload
systemctl enable aistock-api aistock-monitor
systemctl start aistock-api aistock-monitor
systemctl status aistock-api aistock-monitor
```

## 4. 8个选股策略详情

### 4.1 策略一览

| 策略ID | 中文名 | 数据源 | 评分公式 | Top N |
|--------|--------|--------|----------|-------|
| `short_term_hunter` | 短炒猎手 | 新浪A股排行 | 涨幅×0.30 + 金额×0.25 + 盘中位置×0.20 + 实体比×0.15 + 量比×0.10 | 40 |
| `growth_pioneer` | 成长先锋 | 东方财富F10 | PEG×0.25 + 利润增长×0.25 + 毛利率×0.20 + ROE×0.15 + 负债率×0.15 | 30 |
| `stable_fortress` | 稳健堡垒 | F10+腾讯行情 | ROE×0.25 + 股息率×0.20 + PE分位×0.20 + 利息保障×0.20 + 负债率×0.15 | 30 |
| `speed_assassin` | A股游资 | 新浪+腾讯 | 打板确认(≥7%) / 尾盘潜伏(2%-6%) × 板块热度×0.25 + 封板强度×0.20 | 25 |
| `speed_assassin_b` | A股游资B | 同+AI增强 | 游资评分 + App端AI二次分析 | 20 |
| `overnight_navigator` | 隔夜导航 | 新浪+腾讯 | 收盘形态×0.25 + 方向预判×0.25 + 板块持续×0.20 + 换手率×0.15 + 市值×0.15 | 20 |
| `koi_picker` | 锦鲤选股 | 四因子融合 | 资金(30%) + 趋势(25%) + 基本面(25%) + 事件(20%) | 25 |
| `koi_picker_b` | 锦鲤选股B | 四因子深度+AI | 锦鲤评分 + 穿透式财务 + App端AI增强 + 五红线风控 | 15 |

### 4.2 公共一票否决规则

所有策略共享以下硬性过滤条件：
- ST / *ST 股票
- 北交所 (bj 前缀)
- 夕阳行业: 钢铁、煤炭、水泥、造纸、玻纤、氯碱、纯碱、PVC
- 新股 (N开头且名称<6字)
- 股价低于 2 元
- 资产负债率 > 80%
- 低毛利率且利润负增长

### 4.3 锦鲤选股B（量化私募级）

评分体系：**财务40% + 资金40% + 技术20%**
红线风控：5道一票否决 (ST、夕阳行业、高负债、低毛利负增长、低价股)
接入云端AI模型进行深度穿透分析

## 7. 三大投资模块规则

### 7.1 模块对比

| 模块 | 单只上限 | 组合上限 | 止盈 | 止损 | 周期 |
|------|---------|---------|------|------|------|
| 极速投资 | ¥20,000 | ¥120,000 (6只) | 动态 | -4% | T+1买入/T+2卖出 |
| 热点投资 | ¥30,000 | ¥90,000 (3只) | +10% | -4% | 最长5个交易日 |
| 轻量投资 | ¥3,333 | ¥10,000 (3只) | +10% | -4% | 继承热点 |

### 7.2 极速投资 (Speed Investment)

- **选股时间**: 交易日20:00自动选股 (A股游资前3 + 隔夜导航前3 = 6只)
- **买入**: T日09:30按开盘价买入
- **卖出**: T+1日09:30按实时价卖出
- **日期显示**: 卡片显示选股日(createTime)，非建仓日(buyDate)
- **结算记录**: 按 date+portfolioId 去重

### 7.3 热点投资 (Hot Investment)

- **选股**: 从S/A级热点追踪一键建仓 (3只)
- **止盈**: 持仓盈利达到+10%自动卖出
- **止损**: 持仓亏损达-4%自动卖出
- **强制平仓**: 第5个交易日14:55未止盈止损则强制清仓
  - 超过5个交易日（第6天起）：09:30开盘即执行强制平仓
  - 盘后（15:05-次日09:30）：检测到超期持仓仍可触发平仓
- **T+1限制**: 买入当日不能卖出

### 7.4 轻量投资 (Lite Investment)

- **继承**: 完全复用热点投资所有逻辑
- **差异**: 单只上限¥3,333, 总上限¥10,000; 独立存储key (`lite_` 前缀)

### 7.5 收益统计 (Expert Performance)

- 追踪专家选股策略的历史收益
- 每日明细表追踪股票涨跌幅
- 支持手动强制结算和自动结算
- 多股涨跌幅追踪 (支持同时追踪多只股票)
